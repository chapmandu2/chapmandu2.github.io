Bioinformatics Tutorials

Phil Chapman

2017/11/06

I recorded a number of video tutorials before I left CRUK-MI on how to use various resources. These can be found on their own YouTube Channel or via the list below:

  • Introduction to cell line data: Overview of data available and an explanation of different genomics data types and what they can be used for.
  • Using IGV To Explore RNAseq Data: How to use the Broad Integrated Genomics Vieer to identify mutations and their consequences in RNAseq data
  • Using the CCLE Genomics Explorer Shiny App To Identify Cell Lines : How to explore public domain cell line genomic data to identify suitable cell lines for further experimentation.
  • Using IGV To Understand Cell Line Copy Number Data: How to explore copy number information at the genomic level using IGV
  • Using ENSEMBL To Find A Gene Sequence: How to use ENSEMBL to understand different transcript variants for a gene and choose the most appropriate one for experimentation.

  • IncucyteDRC Demo: Demonstration of the Incucyte DRC shiny application

  • cBioPortal Demo Part 1 and Part 2: Demonstration of how to use the cBioPortal to explore large genomics cohorts such as TCGA
